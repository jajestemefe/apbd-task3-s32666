# APBD Task 3: LINQ Lab

In-memory university data analysis using C# and LINQ.

## How to Run

1. Clone this repository
2. Open the solution in Rider or Visual Studio
3. Ensure the .NET 8 SDK is installed
4. Run the project and choose a task from the interactive menu

## Project Layout

* `Models` - Core entities (Student, Lecturer, Course, Enrollment)
* `Data` - Static seeded university dataset
* `Exercises` - Core LINQ query implementations

## Tasks Overview

Features 16 standard tasks and 4 bonus challenges focusing on data filtering, grouping, inner joins, and aggregations.
